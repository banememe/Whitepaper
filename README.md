3. Liquidity & Security

Upon reaching 100% of the bonding curve, $BNM automatically migrates to Raydium DEX. All accumulated SOL and the remaining 200,000,000 $BNM tokens (20% of total supply) are deposited into the Liquidity Pool. The LP tokens are permanently burned, ensuring that the liquidity is locked forever.

This automatic and transparent mechanism provides 100% security for all $BNM holders and makes the project completely immune to rug-pulls.
